# basic_react_express_app

Prerequisites:

   1. install nodeJS
   
  How to install nodeJS in ubuntu:
    1. open the terminal
    2. write the followin commands => 
    	$ sudo apt-get install curl
	$ curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
	$ sudo apt-get install nodejs


Client:

        cd client
        npm install
        npm start

API:

        cd api
        npm install
        npm start
